Model 3

Model 3는 듀얼 모터 상시 사륜구동(AWD), 20인치 Überturbine 휠 및 퍼포먼스 브레이크 옵션을 장착
하여 어떤 기후 조건에서도 완벽하게 제어할 수 있습니다. 카본 파이버 스포일러가 고속 주행 시 안정성을
향상시켜 Model 3의 0-100 km/h* 도달시간이 약 3.3초에 불과합니다.

Model Y

모두를 위한 공간 Model Y는 최대 5명이 탑승하고 장비도 적재할 수
있을 만큼 다재다능합니다. 각 2열 좌석은 독립적으로 폴딩되어 스키용품, 가구 및 여행용 가방 등을 유연하 게
보관할 수 있습니다. 후면 트렁크는 트렁크 깊은 곳까지 문이 열려 쉽고 빠르게 짐을 적재할 수 있습니다.

Model X

Model X는 개발 단계부터 전기차로 설계되어 견고한 구조와 차체 하부에 장착된 배터리팩으로
탑승자 상해 확률 및 차량 전복 위험을 최대한 낮출 수 있습니다.
모든 Model X는 Tesla의 최신 액티브 세이프티 기능인 자동 긴급 제동 등이 추가 비용 없이 기본으로 제공됩니다.

Model S

Model S Plaid는 현존하는 양산 차량 중 가장 빠른 가속력을 자랑하는 차량 중 하나입니다.
모든 Model S는 업데이트된 배터리 구조를 통해 반복적인 트랙 주행에도 성능 저하가 일어나지 않습니다.
